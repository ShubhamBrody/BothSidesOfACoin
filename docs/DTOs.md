# Data Transfer Objects (DTOs)
## BothSidesOfACoin — Complete DTO Specification

All DTOs are defined as Pydantic v2 models (Python backend) and TypeScript interfaces (frontend).

## 1. Authentication DTOs

### Request DTOs

```python
# ── Registration ──
class RegisterRequest(BaseModel):
    email: EmailStr
    password: str = Field(min_length=8, max_length=128)
    full_name: str = Field(min_length=2, max_length=100)
    
    @field_validator("password")
    def validate_password(cls, v):
        """At least 1 uppercase, 1 lowercase, 1 digit, 1 special char"""
        ...

# ── Login ──
class LoginRequest(BaseModel):
    email: EmailStr
    password: str

# ── OAuth ──
class OAuthCallbackRequest(BaseModel):
    code: str
    state: str
    provider: Literal["google", "github"]

# ── Password Reset ──
class ForgotPasswordRequest(BaseModel):
    email: EmailStr

class ResetPasswordRequest(BaseModel):
    token: str
    new_password: str = Field(min_length=8, max_length=128)

# ── Token Refresh ──
class RefreshTokenRequest(BaseModel):
    refresh_token: str
```

```python
class AuthTokenResponse(BaseModel):
```

## 3. AI Debate DTOs

### Request DTOs

```python
class DebateStartRequest(BaseModel):
    topic_id: UUID | None = None
    custom_topic: str | None = Field(default=None, max_length=500)
    
    @model_validator(mode="after")
    def validate_topic(self):
        if not self.topic_id and not self.custom_topic:
            raise ValueError("Either topic_id or custom_topic is required")
        return self

class DebateMessageRequest(BaseModel):
    content: str = Field(min_length=1, max_length=2000)
    target_agent: Literal["all", "left", "right", "neutral"] = "all"
```

### Response DTOs

```python
class DebateSessionResponse(BaseModel):

        "⚠️ AI-generated content may contain inaccuracies. "

class WSDebateEvent(BaseModel):
    type: Literal["token", "message_complete", "error", "session_end"]

    content: str

    timestamp: datetime
```

## 9. AI Agent Internal DTOs (Backend Only)

These DTOs are used internally by AutoGen agents and are not exposed to the frontend.

```python
# ── Orchestrator ──
class ExecutionPlan(BaseModel):
    request_id: str
    request_type: str
    steps: list["ExecutionStep"]
    created_at: datetime

class ExecutionStep(BaseModel):
    step_number: int
    agent_name: str
    description: str
    input_mapping: dict[str, str]  # Maps from previous step outputs
    priority: Literal["required", "optional"]
    timeout_seconds: int = 30
    max_retries: int = 2

# ── Agent Results ──
class AgentResult(BaseModel):
    agent_name: str
    step_number: int
    success: bool
    output: dict  # Flexible output per agent
    error: str | None = None
    execution_time_ms: int
    model_used: str
    tokens_used: int

# ── News Collector ──
class RawArticle(BaseModel):
    title: str
    content: str
    url: str
    source_name: str
    author: str | None
    published_at: datetime | None
    image_url: str | None

# ── Bias Classifier Output ──
class BiasClassification(BaseModel):

    bias_label: BiasLabel
    confidence: float  # 0.0-1.0
    reasoning: str
```
